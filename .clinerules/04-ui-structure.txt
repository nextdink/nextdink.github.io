APP STRUCTURE

Bottom Navigation (3 Tabs):

2. Discover

--------------------------------------------------
1. HOME (Mission Control)
--------------------------------------------------

Purpose:
Quickly see what you're playing next.

Sections:

A. Next Upcoming Event (Large Card)
- Event name
- Date + time
- Venue name
- Teams (3 / 4 teams) or Spots (6 / 8 spots)
- Your status (Joined / Waitlisted)

B. Other Upcoming Events

C. Invitations (if any)
- Event card with Accept / Decline buttons

D. Events You Own

E. Quick Actions
- Create Event

--------------------------------------------------
2. DISCOVER (Public Events)
--------------------------------------------------

Top Filters:
- City selector
- Date filter

Event Card Layout:
- Event name
- Date
- Time
- Venue name
- Teams filled (3 / 4) or Spots (6 / 8)
- Open spots indicator (2 open slots)
- Distance (if location enabled)
- Join button

Empty State:
- "No events found"
- CTA: Create Event

No map view for MVP.

--------------------------------------------------
3. PROFILE
--------------------------------------------------

Top:
- Profile photo
- Display name

Sections:
- My Events (Owned / Joined)
- Lists
- Settings
  - Account settings
  - Theme toggle
  - Logout

- Modal opens: "Build Your Team"
- First slot: You (auto-filled)
- Additional slots (based on teamSize):
  - Option A: Enter guest name
  - Option B: "Looking for partner" (open slot)

--------------------------------------------------
LIST DETAIL PAGE
--------------------------------------------------

- List name
- Member count
- Member list (with profile photos)

Owner can:
- Add/remove members
- Promote admins
- Demote admins

Admins can:
- Add/remove members

Only Owner can remove admins.

Lists are private.
No chat.

--------------------------------------------------
LEAVING BEHAVIOR
--------------------------------------------------

**Team Captain leaves:**
- Confirmation modal: "This will remove your entire team from the event"
- Entire team removed
- If waitlisted teams exist â†’ first one auto-promoted

**Team Member leaves (post-MVP when users can be added to teams):**
- Their slot becomes "Open" (looking for partner)
- Team stays registered

--------------------------------------------------
OWNER/ADMIN ACTIONS
--------------------------------------------------

Owner sees on Event Detail:
- Edit Event
- Cancel Event
- Remove Team (on each team card)

Admins see:
- Edit Event
- Invite Players
- Remove Team

--------------------------------------------------
DESIGN PRINCIPLES
--------------------------------------------------

- Clean, neutral color palette
- One primary brand color
- Strong typography hierarchy
- No gradients
- No social metrics
- No chat UI
- No ratings
- No engagement mechanics
- Fast creation flow
- Clear capacity visibility
- Clear team structure
- Clear ownership visibility
- Clear state transitions
