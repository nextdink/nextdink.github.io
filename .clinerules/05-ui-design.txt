UI Design

Next Dink – Visual Design System (MVP)

Design Philosophy: Simple, flat, aesthetic. Clean lines, generous whitespace, calm and functional.

---

DESIGN PRINCIPLES

- Flat design — no shadows, no gradients, no 3D effects
- Minimal color palette — neutrals with one accent color
- Clean borders — subtle 1px borders instead of shadows
- Generous whitespace — breathing room between elements
- Simple shapes — consistent 8px border radius
- Calm aesthetic — understated, functional elegance
- Mobile-first — optimized for touch interfaces

---

COLOR PALETTE

Using Tailwind's color system with teal as the accent color.

Accent (Brand):
- Light mode: teal-600 (#0d9488)
- Dark mode: teal-400 (#2dd4bf)

Backgrounds:
- Light mode: white, slate-50
- Dark mode: slate-950, slate-900

Text:
- Light mode primary: slate-900
- Light mode secondary: slate-500
- Dark mode primary: slate-50
- Dark mode secondary: slate-400

Borders:
- Light mode: slate-200
- Dark mode: slate-800

Semantic Colors:
- Success: emerald-600 / emerald-400
- Warning: amber-600 / amber-400
- Error: red-600 / red-400

---

TYPOGRAPHY

Font Family: Inter
Fallback: system-ui, -apple-system, sans-serif

Load via Google Fonts or Fontsource:
- Weights: 400 (normal), 500 (medium), 600 (semibold)

Type Scale:
- Page title: text-2xl (24px) font-semibold
- Section title: text-lg (18px) font-semibold
- Card title: text-base (16px) font-medium
- Body: text-base (16px) font-normal
- Secondary: text-sm (14px) font-normal text-slate-500
- Caption: text-xs (12px) font-medium

Line Height:
- All text: leading-normal (1.5)

---

SPACING

Base unit: 4px (Tailwind default)

Layout spacing:
- Page padding: px-4 py-6
- Section gap: space-y-6
- Card padding: p-4
- Component gap: gap-3
- Inline gap: gap-2

Container:
- Max width: max-w-lg (512px)
- Centered: mx-auto

---

DARK MODE

Implementation:
- Tailwind dark: variant
- Detect system preference with prefers-color-scheme
- Store user override in localStorage
- Apply dark class to html element

User settings options:
- System (default)
- Light
- Dark

---

LOADING STATES

Simple approach — no skeleton screens.

Page loading:
- Centered spinner (teal color)
- animate-spin

Button loading:
- Replace text with spinner
- Maintain button width

Inline loading:
- Small spinner next to text

---

EMPTY STATES

Centered layout:
- Icon (w-12 h-12, slate-300)
- Title (text-lg font-medium)
- Description (text-sm text-slate-500)
- Optional CTA button

---

TRANSITIONS

Keep minimal:
- Color transitions: transition-colors duration-150
- No scale or transform effects
- No elaborate animations

Respect prefers-reduced-motion:
- Disable non-essential animations

---

ACCESSIBILITY

- Touch targets: minimum 44x44px
- Color contrast: WCAG AA compliant
- Focus states: outline-none ring-2 ring-teal-600 ring-offset-2
- Semantic HTML elements
- ARIA labels where needed

---

RESPONSIVE

Mobile-first using Tailwind breakpoints:
- Base: < 640px (primary target)
- sm: 640px+
- md: 768px+

No complex desktop layouts for MVP.
Optimize for single-column mobile view.
